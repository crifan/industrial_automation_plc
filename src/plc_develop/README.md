## PLC系统中，PLC编程人员如何下载代码到PLC硬件中

PLC编程人员，在电脑编写好PLC代码后，通过电缆把电脑连接到PLC上，然后下载代码到PLC硬件中：

![plc_download_code_into_hardware](../assets/img/plc_download_code_into_hardware.jpg)

即：

* 电脑：编程设备
* 人：用电脑写PLC代码的人
* 电脑连接PLC：用电缆
* PLC：下载到PLC的CPU中
